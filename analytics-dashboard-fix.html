Analytics Dashboard Fix

I've fixed the JSX syntax error in the AnalyticsDashboard component by correcting a missing closing tag.

Issue Fixed: The component had a missing closing tag for a motion.div element which was causing the Babel parser to fail.
Solution: Replaced the incorrect motion.div closing tag with a proper div closing tag to match the opening element.
Impact: The advanced analytics page should now load correctly without the JSX parsing error.

To view the advanced analytics page, please visit:

http://localhost:3009/advanced-analytics
